升级 flutter sdk 并 同时更新Flutter SDK和你的flutter项目依赖包:

```
flutter upgrade
```
更新项目依赖包：

```
flutter packages get 获取项目所有的依赖包
flutter packages upgrade 获取项目所有依赖包的最新版本
```
